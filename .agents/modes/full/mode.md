# Agent Mode: Full

Use when you want a project to behave like a mature agentic development
environment.

## Enabled

- project-onboarding
- product discovery
- implement
- debug
- review
- docs
- scm
- orchestrate
- ultrawork
- ralph
- independent-verifier
- eval records
- proposed-rules promotion loop

## Expectations

- Use subagents for broad work.
- Use ultrawork for risky multi-file implementation.
- Use ralph when the user gives mechanical completion criteria and wants repeat-until-done behavior.
- Keep feature logs and case records updated.
- Propose reusable rule improvements instead of silently editing permanent rules.
