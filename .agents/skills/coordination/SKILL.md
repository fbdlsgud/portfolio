# Coordination Skill

Use when multiple subagents or workflows are needed.

## Process

1. Classify task difficulty using `_shared/core/difficulty-guide.md`.
2. Select workflow.
3. Route to the fewest useful agents.
4. Check `resources/role-boundaries.md` before assigning conflicting ownership.
5. Track validation and docs/ai records.
6. Escalate to ultrawork or ralph only when justified.
