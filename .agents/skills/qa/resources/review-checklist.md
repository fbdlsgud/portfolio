# Review Checklist

## Security

- auth and authorization
- secret handling
- input validation
- injection risks
- sensitive logging

## Correctness

- edge cases
- null/empty states
- race conditions
- data consistency
- error handling

## Performance

- expensive loops
- unnecessary re-renders
- N+1 queries
- large payloads
- blocking operations

## Accessibility

- labels
- keyboard navigation
- focus handling
- contrast
- responsive text

## Maintainability

- naming
- boundaries
- duplication
- hidden coupling
- test gaps
