# Dev Workflow Skill

Use for setup commands, validation pipelines, scripts, CI, and local developer
workflow.

## Process

1. Read `.ai-local/project-profile.yaml`.
2. Inspect package/build/task files.
3. Prefer project-defined commands.
4. Avoid installing dependencies unless needed.
5. Report command failures with exact next steps.
