# Mobile Skill

Use for React Native, Flutter, native iOS, native Android, device permissions,
mobile UI, and mobile state flows.

## Process

1. Confirm the mobile stack.
2. Read project conventions and active profile.
3. Inspect existing screen/state/navigation patterns.
4. Implement scoped mobile changes.
5. Consider permissions, offline, loading, and error states.
6. Validate on simulator/device when practical.
7. Record meaningful work in `docs/ai/feature-log/`.
