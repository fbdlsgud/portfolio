# Memory Protocol

This kit uses project-local records instead of hidden long-term memory.

## Write Records To

- `docs/ai/feature-log/`
- `docs/ai/error-cases/`
- `docs/ai/review-cases/`
- `docs/ai/decisions/`
- `docs/ai/proposed-rules/`
- `docs/ai/evals/`

## Rules

- Records must be concrete and dated.
- Current code beats stale records.
- Do not write secrets.
- Do not promote proposed rules without review.
