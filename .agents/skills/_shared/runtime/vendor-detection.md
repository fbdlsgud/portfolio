# Vendor Detection

This kit is Codex-first.

Assume Codex unless the user explicitly says they are using another coding
agent.

## Codex Path

- read `AGENTS.md`
- use `.codex/agents/*.toml` for subagent roles
- use `.codex/hooks.json` for hooks
- use `.agents/workflows/*.md` for workflow instructions

## Other Agents

Other agents may still read these files, but behavior is not guaranteed.
