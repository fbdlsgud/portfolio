# Eval Workflow

Use after changing the kit or project AI setup.

## Steps

1. Choose eval cases from `evals/`.
2. Run the same prompt before and after the setup change when possible.
3. Score criteria 0-2.
4. Record results in `docs/ai/evals/`.
5. Promote or revert setup changes based on evidence.

## Output

```text
Eval cases:
Scores:
Regressions:
Improvements:
Decision:
```
