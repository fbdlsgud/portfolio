# Work Workflow

Use for step-by-step execution with remediation, lighter than Ralph and heavier
than direct implementation.

## Steps

1. Read project context.
2. Define scope.
3. Implement the smallest useful slice.
4. Run validation.
5. If validation fails, fix root cause once.
6. Run review.
7. Update docs/ai records.

## Gate

- [ ] requested behavior implemented
- [ ] validation attempted
- [ ] review completed for risky changes
- [ ] docs/ai updated when meaningful
