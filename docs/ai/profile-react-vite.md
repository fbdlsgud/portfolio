# React Vite Profile

## Assumptions

- Frontend source lives in `src/`.
- Use existing component and styling conventions before introducing new ones.
- Keep components small and focused.
- Use semantic HTML and accessible controls.
- Run lint, typecheck, and build before completion when available.

## Agent Notes

- For visual changes, verify in browser when practical.
- Record meaningful UI features in `docs/ai/feature-log/`.
- If the project uses a component library, prefer it over custom components.
